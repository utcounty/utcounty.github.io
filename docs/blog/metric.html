The Precinct

The smallest political unit in the election. It is a specific geographic shape (polygon). Everyone living inside this shape votes at the same location (or gets the same ballot style).

Vote-for-N Contests

While most races restrict voters to a single vote per candidate (vote_for = 1), some races like City Council races allow voters to vote for multiple candidates (vote_for > 1).

When vote_for > 1, contest_total_votes counts selections, not ballots, contest_total_votes can exceed precinct_ballots_cast and make rolloff_rate look negative.

max possible selections = precinct_ballots_cast × vote_for
